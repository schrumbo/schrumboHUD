# SchrumboHUD
SchrumboHUD is a highly customizable Minecraft mod that permanently displays the player's inventory and/or armor on the screen.

## Mod Features
- **Inventory HUD**
- **Armor HUD**
- **Peek Mode** - hold keybind to temporarily show HUD

You can either use one of the color presets or change the colors yourself.

## Keybinds
| Action | Default Key |
|--------|-------------|
| Open Config | Right Shift |
| Toggle Show Always | L |
| Peek HUD | Unbound |

You can also use `/shud` to open the config.
