# Mod Features

## Controls
- **Toggle HUD**: `L` (default)
- **Toggle ClickGUI**: `Right Shift` (default)

## Customization Options
### Colors
- **Background**
- **Border**
- **Text**
- **Accent**
- **Slots**

Opacity of all colors may also be changed
